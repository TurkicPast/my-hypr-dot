### ⌨️ Keybindings (Wurh's Setup)

| Keybind | Action |
| :--- | :--- |
| `SUPER + Enter` | Open Terminal (Kitty) |
| `SUPER + Q` | Close Active Window |
| `SUPER + E` | File Manager (Thunar) |
| `SUPER + B` | Web Browser (Firefox) |
| `SUPER + D` | App Launcher (Rofi) |
| `SUPER + F / SPACE` | Toggle Floating Mode |
| `SUPER + R` | Reload Hyprland |
| `Print` | Screenshot Fullscreen (Clipboard) |
| `SUPER + Shift + S` | Screenshot Area (Clipboard) |
| `Brightness Keys` | Control Display Brightness (+/- 5%) |
| `SUPER + Arrows` | Move Focus (Left/Right/Up/Down) |
| `CTRL + ALT + DEL` | Exit Hyprland |
